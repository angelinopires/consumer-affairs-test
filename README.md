## 🤖 Prerequisites
* Node version used: **14.17.6**
* NPM version used: **6.14.15**

## ▶️ Getting start

Install dependencies:

```bash
$ npm install
```

Run as a development server:

```bash
$ npm run dev
```

Open this link on your favorite browser [http://localhost:8080](http://localhost:8080).
